Windows 7 换IP教程（换IP都不会，我也是醉了）

  • Windows
  • 教程

一般地，要连上校园网都要自行设置静态IP，每栋楼如何设置，网管中心给出了规定，大家可以通过这个网页查询到具体的规定：学生公寓IP查询，务必断开 麦芒网络 再尝试打开该网页。

最近，有些新建的楼已经开通了 DHCP 服务——即动态 IP 服务，但是我们发现，动态IP对 麦芒网络 并不友好，所以我们建议大家还是使用静态IP。

需要说明的是，所谓的DHCP服务，就是本机不再需要设置静态 IP 地址，IP 地址的分配由交换机负责。

而自行设置静态IP地址，只是将IP地址的分配由个人接管，这与交换机是否有DHCP功能并不冲突。

什么是DHCP？查看 DHCP_百度百科 了解。

至于我们为什么需要换 IP？我一直强调，内网是连上麦芒网络的基础！所以换 IP，就是为了连上内网！

好的，需要说明的就这些了，下面以 Windows 7 系统为例，为小白们介绍如何换 IP 。

首先 右键 任务栏右下角的 网络图标，单击 打开网络和共享中心。

在网络和共享中心页面的左侧，找到 更改适配器设置，单击进入。

这时你会看到以下图标，本地连接为你的有线网卡，无线网络连接为你的无线网卡，麦芒网络是你设置的连接（不用管）。

如果你的电脑是用网线连上网络的，右键本地连接；如果你的电脑是连接 Wifi 上网的，右键无线网络连接。

这里我是用网线连上网络的，所以我右键本地连接，选择属性。

在 本地连接 属性 - 网络 选项下，找到 Internet 协议版本4（TCP/IPv4），双击进入。

进入 Internet 协议版本4（TCP/IPv4）属性 选项卡，这里解释一下各个选项：

  1. 自动获得 IP 地址：这是设置动态 IP 的选项，不勾选。
  2. 使用下面的 IP 地址：这是设置静态 IP 的选项，勾选之，然后填写：
    • IP 地址和默认网关：根据各楼的规定填写，不知道具体规定的请看 学生公寓IP查询 ；
    • 子网掩码：填写 255.255.255.0 即可；
    • DNS 服务器地址：根据图示填写即可。

填写完成后，连续单击 确定 就完成了静态 IP 的设置。
