【必读】加入麦芒网络，享受更加简单的网络

之前使用的博客平台估计是要倒了，从去年服务器偶尔抽风，到现在直接没法登录了，无奈只能新开一个博客。

本博客跟之前一样，将包含 Windows、Android、OS X、iOS等主流平台的使用教程，我们致力于让网络更加简单便捷好用，惠及每一位客户。

关于 麦芒 的服务，这里需要注意几点：

  1. 我们的帐号一律采用实名登记，不允许私自转让、借用，如需购买请联系客服，不要去跳蚤市场求转让；转让或借用帐号的行为，一经发现，做停号处理。

  2. 连接 麦芒 的服务，必须能接入内网，内网是基础。

  3. 因静态IP的缘故，您的设备所设置的IP有可能与他人的相同，这称为IP冲突，会造成服务间歇性掉线，或者无网络服务，这时更改IP后可以解决。

  4. 一个账号，在同一时间只能登陆一台设备，重复登陆会报错。

  5. 账号到期后，可以登录，但是没有流量，这是请联系客服续费。

  6. 连上 麦芒 的服务之后，内网的 BT 就不能上传下载了，请根据这篇 Windows 和 Linux 系统连上麦芒网络，不能下载内网BT资源的解决办法 教程进行设置解决该问题。

  7. 下载限时：规定如下图所示,为保证大家网速顺畅，请各位遵守该规定。

    • 为保证网页浏览、聊天、游戏等的流畅，我们对下载行为采取了严格的限时策略。
    • 凡在限制下载的时段，用户不允许下载视频、镜像等大文件（累计不超过200M）。
    • 用户的下载行为会受服务器流量监控，凡下载者会被自动调为限速用户，经常下载者不予续费。

8.如遇到不能解决的问题，请及时联系客服。

麦芒 使用教程如下：

  1. 在 Windows 7 中使用 麦芒网络 教程
  2. 在 Windows 8.1 中使用 麦芒网络 教程
  3. Windows 和 Linux 系统连上麦芒网络，不能下载内网BT资源的解决办法
  4. Windows 7 换IP教程（换IP都不会，我也是醉了）
  5. 在 Android ( MIUI ) 中使用 麦芒网络 教程
  6. 在 iOS 7.1.1 ( iPhone ) 中使用 麦芒网络 教程
  7. 在 OS X 10.8 ML 中使用 麦芒网络 教程
  8. 还需要啥？留言给我！
