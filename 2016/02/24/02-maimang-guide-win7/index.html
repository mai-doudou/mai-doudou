在 Windows 7 中使用 麦芒网络 教程

  • Windows
  • 教程

一般的，使用系统自带的连接选项，就可以使用 麦芒网络 ，而使用客户端，会出现这样、那样不可预知的问题，所以不推荐使用客户端，下面我就将教会大家使用 Windows 7 自带的连接选项，轻松实现连接 麦芒网络。

OK，首先必须确定你的电脑已经连上了内网，并能够打开内网的一些网站，然后开始下面的步骤。

依次打开 控制面板 \ 网络和 Internet \ 网络和共享中心 ，进入网络和共享中心，或者点击任务栏中的 网络 图标，然后单击 打开网络和共享中心，进入网络和共享中心；

在 网络和共享中心 中，单击 设置新的连接或网络，在弹出窗口中，选择 连接到工作区，单击 下一步；

如果出现下面的窗口，询问 您想使用一个已有的连接吗？ 选择 否，创建新的连接 ，单击 下一步 继续；

选择 使用我的 Internet 连接(VPN)；

如果出现下面的窗口，询问 您想在继续之前设置 Internet 连接吗？,选择 我将稍后设置 Internet 连接；

下面填写 Internet 地址，即服务器地址，具体请询问客服，目标名称 填写 麦芒网络，单击 下一步；

下面填写 用户名 和 密码，勾选 记住此密码（这个看你个人喜好） ，单击 创建；

这里是重点！到了下面这个界面，务必单击 跳过，因为需要进一步设置，才能使用；

单击 跳过 之后，单击 关闭，然后进行下一步设置；

同第一步，打开 控制面板 \ 网络和 Internet \ 网络和共享中心 ，进入网络和共享中心，或者点击任务栏中的 网络 图标，然后单击 打开网络和共享中心，进入网络和共享中心，单击左侧的 更改适配器设置，进入下一步；

找到你刚刚创建的 麦芒网络，右键之，单击 属性，在弹出的 属性选项卡 的 安全 选项下，将 V-P-N 类型 设置为 点对-点隧-道协-议(P-P-T-P)，单击 确定，完成设置，到这里，就完成了连接的创建工作；

下面就可以连接了， 单击任务栏中的 网络 图标， 点开 麦芒网络， 单击 连接， 会弹出 账户和密码填写窗口，因为创建 麦芒网络 的时候，已经填写了账户和密码，所以这里直接单击 连接，就会开始连接服务器，连接成功之后，再次单击任务栏中的 网络 图标，可以看到 麦芒网络 显示 已连接，Enjoy it ！

以后，只需最后一步连接工作，就可以使用 麦芒网络 ，无需重复创建连接的工作。下面说两条tips ，方便大家的使用（这两条tips，不看也没关系） 。

首先是进一步设置，使 麦芒网络 更易使用。如图，右键 麦芒网络，点击 属性，弹出 麦芒网络 属性 窗口；在 选项 下，不勾选 连接时显示进度 和 提示名称、密码和证书等，这样下次连接时，就不会弹出上图中的账户、密码填写窗口，并且不会显示连接进度，总的连接步骤只有三次鼠标单击，比较简洁；建议将 重拨间隔 调小，这样在意外断开连接之后，就可以很快重新连上，当然，你得先勾选上 断线重拨 。

第二点， 如果你的 拨号和 V-P-N 选项卡 中的连接数量太多,可以在 控制面板 \ 网络和 Internet \ 网络和共享中心 \ 更改适配器设置 中，将多余的连接选中，右键，删除。

差不多就这些内容了，再补充几点：

  • 麦芒网络 提供的账号，同一时间只允许一台设备登陆，如果连接过程中，出现691错误提示，请查看是否还有其他设备处于连接状态；
  • 使用 360 软件，易引起网络卡顿、掉线，故不推荐使用 360；
  • 网络间歇性断开，极有可能是 IP 冲突引起的，请换 IP 后重连。不会换 IP？参考这篇 Windows 7 换IP教程 吧。
