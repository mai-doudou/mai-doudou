在 Windows 8.1 中使用 麦芒网络 教程

  • Windows
  • 教程

Windows 8 的连接方法，与 Windows 7 的连接方法 会有一定区别，所以需要单独写一篇 Windows 8 的教程。

但是 Windows 10 的连接方法与 Windows 8 相似，由于本人还未升级 Windows 10，没法截图，所以使用 Windows 10 的亲们，就对着这篇看吧，设置很简单的。

OK，首先必须确定你的电脑已经连上了内网，并能够打开内网的一些网站，然后开始下面的步骤。

依次打开 控制面板 \ 网络和 Internet \ 网络和共享中心 ，进入网络和共享中心，或者点击任务栏中的 网络 图标，然后单击 打开网络和共享中心，进入网络和共享中心；

在 网络和共享中心 中，单击 设置新的连接或网络；

在弹出窗口中，选择 连接到工作区，单击 下一步；

选择 使用我的 Internet 连接（V-P-N）；

如果出现下面的窗口，询问“您想在继续之前设置 Internet 连接吗？”，请选择 我将稍后设置 Internet 连接；

下面填写 Internet 地址，即服务器地址，具体请询问客服，目标名称 请填写 麦芒网络，单击 创建；

右侧弹出网络连接，这时别急着点 连接，还需进一步设置才行；

同第一步，打开 控制面板 \ 网络和 Internet \ 网络和共享中心 ，进入网络和共享中心，或者点击任务栏中的 网络 图标，然后单击 打开网络和共享中心，进入网络和共享中心，单击左侧的 更改适配器设置，进入下一步；

找到你刚刚创建的 麦芒网络，右键之，单击 属性；

在弹出的 属性选项卡 的 安全 选项下，将 V-P-N类型 设置为 点对-点隧-道协-议（PP-TP），并勾选 允许使用这些协议 和 Microsoft CHAP Version 2（MS-CHAP v2）,单击 确定，完成设置，到这里，就完成了连接的创建工作；

下面就可以连接了，单击任务栏中的 网络 图标，在弹出的面板中找到 麦芒网络，单击 连接；

弹出 账户和密码填写窗口，填写你的用户名和密码，单击 连接，就会开始连接服务器；

连接成功后，就可以看到 麦芒网络 显示 已连接，Enjoy it !

以后，只需最后一步连接工作，就可以使用 麦芒网络 了，无需重复创建连接的工作。你也可以进一步设置 麦芒网络 的属性，具体可参考 在 Windows 7 中使用 麦芒网络 教程 文末的 tips，我在这里就不赘述了。

值得吐槽的是，Windows 8.1 中， 麦芒网络 掉线居然不会有弹窗提示！实在搞不到微软的进化轨迹，Windows 8.1 比 Windows 7 差远了。
